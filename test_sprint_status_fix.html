Sprint Status Fix Test

This test verifies that tasks get the correct initial status when created with or without sprint assignments.

Test 1: Create task without sprint assignment

Expected: Task should have status 'backlog'

Test 2: Create task with sprint assignment

Expected: Task should have status 'todo'

Test 3: Create subtask without sprint assignment

Expected: Subtask should have status 'backlog'

Test 4: Create subtask with sprint assignment

Expected: Subtask should have status 'todo'

Test 5: Assign existing task to sprint

Expected: Task should have status 'todo' after assignment
